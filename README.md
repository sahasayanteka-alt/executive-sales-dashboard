# Executive Sales & Profit Analysis Dashboard

## ?? Project Overview
An interactive Power BI dashboard designed for C-suite executives to monitor global sales performance, profitability trends, and product-level growth. This tool transforms raw corporate financial data into real-time actionable insights.

## ?? Key Features & Insights
* **Executive KPI Row:** Instant visibility into Total Revenue ($118.73M), Total Net Profit ($16.89M), and Total Units Sold (1.13M).
* **Interactive Slicers:** Dynamic regional filtering allowing executives to drill down into specific countries (e.g., Germany, Canada) with a single click.
* **Monthly Net Profit Trend:** A clean line analysis tracking seasonal profit spikes and dips to assist with forecasting.
* **Revenue Breakdown by Product:** A horizontal ranking identifying high-performing product lines instantly.

## ?? Design Philosophy (Premium Dark Mode)
* **High Contrast:** Built using a custom dark-mode canvas background to reduce eye strain and give a premium, modern software feel.
* **Visual Cleanliness:** Stripped away redundant default chart axes and text labels to maximize data-ink ratio and improve executive readability.
